Candies price

Two friends Alice and Bob went to a store to buy some candies. There are N candies in a row and you are given the price of candies as an array A.

Each of them (Alice and Bob) is willing to buy all N candies. To buy these candies there are some conditions:

  • You are given K plastic bags to put these candies
  • If ith and jth candies are in the same bag, then candies at index K, where i < k < j (for all values of k, assuming that i < j ) must also be in that bag.
  • No bag (among the K bags) should remain empty.
  • If a bag contains candies from index i to j , the cost of this bag will be A[i] + A[j].

Alice is going to pay the maximum possible price for these candies under such constraints, and Bob is willing to pay the minimum possible price.

Task

Determine the difference between the amounts that Alice and Bob will pay (If they get candies).

Example 

Assumptions

  • N = 4
  • A = [1, 3, 5, 1]
  • K = 2

Approach

If they have K = 2 bags to put these candies then:

  • Alice will put candies in 2 bags as [1, 3] [5, 1]
  • Bob will put candies in 2 bags as [1] [3, 5, 1]
  • Total price of Alice's bags: (1+3) + (5+1) = 10, which is maximum possible.
  • Total price of Bob's bags: (1+1) + (3+1) = 6, which is minimum possible.

Difference of amount = 10 - 6 = 4, so the answer is 4.

Function description 

Complete the diffAmount function provided in the editor. This function takes the following 3 parameters and returns the required amount difference; 

  • N: Represents the number of candies denoted by the length of the array
  • K: Represents the number of bags
  • A: Price of candies  denoted by an array of length N

Input format 

Note: This is the input format that you must use to provide custom input (available above the Compile and Test button).

  • The first line contains an integer T denoting the number of test cases. T also denotes the number of times you have to run the diffAmount function on a different set of inputs.
  • For each test case:
    • The first line contains a single integer N denoting the number of candies.
    • The second line contains integer K denoting the number of bags.
    • The next line contains N space-separated integers denoting the price of candies.

Output format 

For each test case, print the difference of amount that Alice and Bob are willing to pay.
